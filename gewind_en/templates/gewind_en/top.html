Welcome!


This site is a free web app that generates a 3D model of wind turbines that you can open on Google Earth. The file format is KMZ, so that you can open with the desktop version of Google Earth. If you don't have the desktop version of Google Earth installed on your PC yet, you can download it from here.


You can use this web app in any way you like, but we designed this web app for the following two applications.

  • To share and review a wind farm project with its stakeholders before and after the construction
  • To make a simple prediction of the impacts of a wind farm project on landscape in Environmental Impact Assessment

If you have any questions about this site or requests for features of this web app, please contact me via tropicbird38@gmail.com.
